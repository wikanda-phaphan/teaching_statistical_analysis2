## Course Overview

This course covers statistical methods for categorical data, nonparametric techniques, quality control, quantitative forecasting, time series analysis, decision analysis, and index numbers. Students will develop skills in both theoretical understanding and practical application using statistical software.

**Topics covered:**

- Categorical Data Analysis
- Nonparametric Statistics
- Quality Control
- Introduction to Quantitative Forecasting
- Estimation of Seasonal Variation
- Decision Analysis
- Index Numbers
